# 🏫 Ozone High School Website - Simple Guide

Hi! This is a website for our school. Think of it like a digital house:
- **The Red Roof (Header)**: Stays at the top of every room (page).
- **The Teal Floor (Footer)**: Stays at the bottom of every room.
- **The Rooms (HTML files)**: These are the different pages like About, Home, or Admissions.
- **The Paint and Glue (styles.css)**: This is where we decide what colors to use and how big items should be.

---

## 🎨 How to change the colors? (styles.css)

Open the file named `styles.css`. I have put big notes inside it!

- **To change the Red Top**: Look for `header { background-color: red; }` and change **red** to a color like **blue** or **green**.
- **To change the Teal Bottom**: Look for `footer { background-color: teal; }` and change **teal** to something else!

---

## ✍️ How to change the words? (HTML files)

Open any file ending in `.html` (like `Home.html`).

- Look for words that are **not** inside these brackets `< >`. 
- For example, if you see `<h1>Welcome</h1>`, you can change **Welcome** to **Hello!**.
- **Be careful!** Don't change the words inside the `< >` brackets, or the "house" might break!

---

## 📂 Where are the pages?

- `index.html`: The very first screen (Login).
- `Home.html`: The main school page.
- `About.html`: Tells people who we are.
- `Admissions.html`: The new page with the "secret" menu that pops out when you hover over it!
